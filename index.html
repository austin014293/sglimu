GPS 오차 분석 및 개선

❌ 초기 성능 한계

정지 상태: ±5m 수준의 불규칙 튐

고속 주행: 최대 ±15m 오차 발생

  • 멀티패스 페이딩에 의한 수신 좌표 드리프트
  • 터널 전방 차선 구별의 정확도 부족

✅ 해결 및 최적화 기법

차량용 외부 안테나 추가: 위성 개수 6개 → 12개 확보

이동평균 필터 알고리즘: 튀는 노이즈 좌표 자동 마스킹

  • 위성 신호 강도 감시(C/No값 기반 필터링)
  • 오차 반경 1m 내외로 축소 성공

GPS 오차 개선 결과

GPS 위치 측정 오차 감소율 90% 개선
개선 전 (기본 Neo-6M) ± 15 m 차선 감지 불가, 불안정한 좌표 튐
➔
개선 후 (안테나 + 소프트웨어 필터) ± 1.2 m 터널 차선 진입 정밀 타게팅 가능

IMU 센서 (MPU6050) 보정

❌ 자이로 / 가속도 드리프트

• 차량의 주행 진동에 의한 누적 오차 발생

• 정지 상태에서도 중력가속도 분포 드리프트

(가속도 적분 시 누적 오차로 차선 위치 추적 오차가 발생)

✅ 영점 오프셋 및 소프트웨어 필터링

• 부팅 시 500개 샘플 기반 캘리브레이션

• 상보필터(Complementary Filter) 구현

가속도계의 고주파 노이즈 차단 + 자이로계의 저주파 드리프트 보정을 결합해 정확한 피치/롤 각도 획득

LED 제어 방식 변경

❌ 기존 계획 (CAN 제어 시도)

OBD2 포트 제어 통신 장벽

  • 차량 제조사 게이트웨이 보안 필터 탑재
  • 외부 모듈의 CAN 트랜시버 쓰기(Write) 명령 차단
  • 실제 운행 차량 미등 CAN 조작의 잠재적 위험성

✅ 변경 계획 (MOSFET PWM 직접 제어)

독립 구동 외부 LED 미등 모듈 구성

  • ESP32 하드웨어 PWM 제어 라인 독립 설계
  • MOSFET 스위칭 소자로 대용량 전류 안정적 스위칭
  • 진입 거리에 대응하는 3단계 전압(디밍) 제어 가능
